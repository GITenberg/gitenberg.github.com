Welcome to Project GITenberg

Project GITenberg is a Distributed, Collaborative,Trackable and Scriptable fork of Project Gutenberg that leverages the power of the Git version control system and the collaborative potential of Github to collect and archive the most accurate library of free digital books in existience.

.

What is Project Gutenberg?

from the Project Gutenberg website:

Project Gutenberg offers over 40,000 free ebooks: choose among free epub books, free kindle books, download them or read them online.

Why the Fork?

Distributed

Our mission is to create a distributed network of books, a bottom-up archive that can be corrected, pulled, and forked for any purpose

Collaborative

Programmers already have a tried-and-true way to collaborate on projects, Git. Github allows this collaboration to be open and social, allowing anyone with a username to maintain our large collection of ebooks.

Trackable

Leveraging the version control that Git provides, we can see how a work changes over time, providing invaluable data to researchers and interested parties.

Scriptable

Serving books the way we serve code has its benefits. Utilizing Git and the extensive Github Api, you can build epub, integrate the changes of a book into your own personal projects, and more.
